# 🏺 ALQUIMIA VISCERAL | O Sopro da Imagem

![Alquimia Visceral Banner](https://raw.githubusercontent.com/pombagiras/melusine/refs/heads/main/Al%C3%AA%20(3)%20(3)%20(1).png)

> *"Reduzir a matéria para libertar a essência. Comprimir o arquivo para expandir a mensagem."*

Este laboratório digital é uma ferramenta de **Alquimia de Imagens**. Aqui, a tecnologia se curva ao ritual: fotos pesadas são transmutadas em arquivos leves, e o silêncio do visual ganha a voz do "Sopro Vital" — mensagens dinâmicas gravadas diretamente na alma (pixels) da fotografia.

---

## 🌒 A Essência do Projeto

Desenvolvido para o ecossistema do **Portal Visceral / Limbo Sonoro**, este compressor não é apenas funcional; é imersivo. Focado em uma experiência **Mobile First**, ele utiliza feedback háptico (vibrações) para simular o pulsar de um coração a cada etapa da transmutação.

### ⚙️ Funcionalidades Arcanas
*   **Transmutação de Matéria:** Compressão real de JPG, PNG, SVG e GIF com redução de até 90% do peso original.
*   **Sopro Vital (Texto Dinâmico):** Inserção de frases com quebra de linha automática e fusão definitiva nos pixels através de Canvas HTML5.
*   **Memória do Abismo:** Integração total com **Supabase** (Storage e Database) para registro histórico de cada ritual.
*   **Feedback Visceral:** Respostas táteis via API de Vibração para uma navegação orgânica.
*   **Estética Dark Wave:** Interface inspirada no minimalismo industrial e no misticismo das encruzilhadas.

---

## 🛠️ Ferramentas de Invocação

*   **Linguagem:** HTML5, CSS3 (Dark Mode Native), JavaScript (Vanilla).
*   **Altar de Dados:** [Supabase](https://supabase.com/) (PostgreSQL & Bucket Storage).
*   **Processamento:** Canvas API para manipulação de imagem *client-side*.
*   **Hospedagem:** GitHub Pages.
